Welcome to ASIM - An virtual economy SIMulator!

You can perform the following operations:

  • Create and edit redistribution of the economic growth methods
  • Create and edit genetic algorithm fitness functions
  • Create and edit simulation templates
  • Run new simulations
  • Display simulation results
  • Read the documentation

You can also use the upper menu to jump easily to any section of this simulation, no matter what is your current request.

Enjoy!
